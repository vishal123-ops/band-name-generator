# Band Name Generator

A simple Python project that generates a band name based on user input.

## Features
- Takes user input
- Combines city and pet name
- Generates a unique band name

## How to Run

1. Clone the repository
2. Open in PyCharm or any Python IDE
3. Run:

python main.py

## Example

Input:
City: Hyderabad  
Pet: Riser  

Output:
Hyderabad Riser
